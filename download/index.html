Planckoscope tilesets

Below are the various tile-sets used in Planckoscope. If you would like to run a local version, without requiring access to the internet, then you will need to download these. The source code can be downloaded from the Github respoitory.

Each tile-set contains over 5000 images, archived and compressed into both tarballs (.tar.gz) and zip files. While the compressed archives are not too large (typically 10-50 MB), the uncompressed files can take up rather a large amount of space and take considerable time to extract.
