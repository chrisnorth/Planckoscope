Planckoscope tilesets

Below are the various tile-sets used in Planckoscope. If you would like to run a local version, without requiring access to the internet, then you will need to download these. The source code can be downloaded from Github.

Caution: each tile-set contains over 5000 images. While the compressed archives are not too large, the uncompressed files can take up rather a large amount of space.
